# diarias-puxada
Um sistema simples, que consegue calcular a quantidade de diárias de motoristas de carreta, calcular o valor das viagens e emitir avisos automaticos quando o valor estiver abaixo do recomentado.

Irá calcular automaticamente:
 - Diárias;
 - Refeições;
 - Impressões de Manifesto;
 - Avisos de limite crítico.

Usuário:
 - Tela de marcar entrada e saída do galpão;
 - Alem de futuramente marcar chegada e saída da fábrica.

 Admin:
 - Controle de chegada e saída de motoristas e veículos;
 - Horas gastas em deslocamento;
 - Horas gastas em carregamento;
 - Localização de pontos de paradas via API Google Maps.
